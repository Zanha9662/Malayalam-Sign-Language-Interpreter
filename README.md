Malayalam Sign Language Interpreter (Flask)

This is a web-based project that uses Flask and AI to detect hand gestures and convert them into text.

 Features
- Real-time hand gesture detection  
- Web interface using Flask  
- Converts gestures into text output  
- Simple and beginner-friendly AI project  

Technologies Used
- Python   
- Flask   
- OpenCV   
- MediaPipe
- TensorFlow

Installation
 bash
pip install flask opencv-python mediapipe tensorflow
